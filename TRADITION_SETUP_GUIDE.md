# ระบบรายละเอียดประเพณี - Guide & Setup

## 📋 สิ่งที่ทำให้สำเร็จ

### ✅ 1. ประเพณีสามารถคลิกเข้าดูรายละเอียดได้

- ยูเซอร์สามารถกดที่การ์ดประเพณี เพื่อเข้าดูข้อมูลเพิ่มเติม
- มีหน้าใหม่ `tradition_detail.php` เพื่อแสดงข้อมูลประเพณีอย่างละเอียด
- มีการออกแบบให้สวยงามและเป็นมืออาชีพ

### ✅ 2. ระบบวันที่จัดงาน

- เพิ่มเก็บข้อมูลวันที่จัดงานของแต่ละประเพณี
- แสดงวันที่บนการ์ดและหน้าเพิ่มเติม
- สนับสนุนรูปแบบวันที่ที่หลากหลาย (เช่น "22 มกราคม" หรือ "22 January")
- สนับสนุนสถานที่จัดงานและข้อมูลเพิ่มเติม

## 🚀 วิธีการตั้งค่า

### ขั้นตอนที่ 1: Update Database

1. เข้าไปที่ URL: `http://localhost/otop_chanthaburi/setup_tradition_dates.php`
2. ระบบจะเพิ่มคอลัมน์ใหม่ลงในตาราง traditions อัตโนมัติ
3. ดูข้อความสำเร็จที่แสดงบนหน้าจอ

```
✓ Successfully executed: ADD COLUMN `event_date` ...
✓ Successfully executed: ADD COLUMN `event_location` ...
✓ Successfully executed: ADD COLUMN `event_details` ...
✓ Database schema updated successfully!
```

### ขั้นตอนที่ 2: เพิ่มข้อมูลประเพณี (Admin Panel)

1. ไปที่ Admin Dashboard
2. เลือก "จัดการข้อมูลงานประเพณี"
3. เพิ่มหรือแก้ไขประเพณี
4. กรอกข้อมูลใหม่:
   - **วันที่จัดงาน**: เช่น "22 มกราคม ทุกปี" หรือ "22 December"
   - **สถานที่จัดงาน**: เช่น "ลานเลอนาร์ด, จันทบุรี"
   - **ข้อมูลเพิ่มเติม**: รายละเอียดเกี่ยวกับงาน เวลาเปิด-ปิด ฯลฯ
5. บันทึก

### ขั้นตอนที่ 3: ทดสอบ

1. ไปที่หน้า "ประเพณี" ที่เมนูหลัก
2. คลิกที่การ์ดประเพณี
3. ควรเห็นหน้าเพิ่มเติมแบบนี้:
   - ชื่อประเพณี
   - รูปภาพขนาดใหญ่
   - รายละเอียดของประเพณี
   - วันที่จัดงาน (เป็น badge สีเหลือง)
   - สถานที่จัดงาน
   - ข้อมูลเพิ่มเติม
   - ลิงก์ไปยังหน้าอื่น ๆ

## 📁 ไฟล์ที่เพิ่ม/แก้ไข

### ไฟล์ใหม่:
- `setup_tradition_dates.php` - Migration script สำหรับอัพเดท database
- `tradition_detail.php` - หน้าแสดงรายละเอียดประเพณี

### ไฟล์ที่แก้ไข:
- `tradition.php` - เพิ่มลิงก์ไปยัง tradition_detail.php และแสดงวันที่
- `admin_tradition.php` - เพิ่มฟิลด์สำหรับจัดการวันที่และข้อมูลเพิ่มเติม

## 🎨 Features ของหน้า Detail

### Header Section
- ชื่อประเพณี
- ปุ่มกลับไปหน้ากำหนดประเพณี
- Gradient background สีเขียว

### Main Content
- **ภาพประเพณี** - รูปภาพขนาดใหญ่พร้อม shadow effect
- **รายละเอียด** - ข้อความอธิบายประเพณี
- **วันที่จัดงาน** - แสดงเป็นแบดจ์สีเหลือง
- **สถานที่จัดงาน** - แสดงเป็นแบดจ์สีฟ้า
- **ข้อมูลเพิ่มเติม** - กำหนดเอง (ส่วนข้อมูลเพิ่มเติม)

### Related Links
- ลิงก์ไปยังสถานที่ท่องเที่ยว
- ลิงก์ไปยังสินค้า OTOP
- ลิงก์ไปยังประเพณีอื่น ๆ

## 💡 ตัวอย่างข้อมูล

### ประเพณี: งานเก่าแพร่
- **วันที่จัดงาน**: 22 มกราคม ทุกปี
- **สถานที่จัดงาน**: ลานเลอนาร์ด, จันทบุรี
- **ข้อมูลเพิ่มเติม**: งานสมมนาประเพณีล้านช้าง งานแสดงศิลป์พื้นบ้าน เปิดวันที่ 22 มกราคม เวลา 08:00-18:00 น.

## 🔧 Customization

### เปลี่ยนสี
- Header background: `tradition_detail.php` ที่ `.detail-header`
- Badge colors: ดูที่ `.event-date-badge` และ `.location-badge`
- Grid layout: ปรับ `col-lg-7` และ `col-lg-5`

### เปลี่ยนข้อความ
- ส่วนหัว: แก้ไข `<h1 class="display-4">` เป็นชื่อที่ต้องการ
- Labels: แก้ไข `<i class="fas fa-calendar-alt">` ไปยังไอคอนอื่น

## ⚠️ หมายเหตุสำคัญ

1. วันที่จัดงาน ไม่จำเป็นต้องใส่ - ปล่อยว่างได้ถ้ายังไม่มีข้อมูล
2. ข้อมูลเก่าของประเพณียังคง preserved - ไม่มีการลบข้อมูลเก่า
3. สามารถเพิ่มข้อมูลเดิมลงในหน้า detail ได้ทันที
